# 🎓 Student Result Prediction System

This project is a Machine Learning-based web application that predicts a student's **Final Exam Score** based on:

- 📚 Study Hours
- 🏫 Attendance (%)
- 📝 Weekly Test Score

---

## 🚀 Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Linear Regression Model
- Model Evaluation (R², MAE, RMSE)
- Interactive UI using Gradio

---

## 🧠 Tech Stack

- Python 🐍
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
- Gradio

---

## 📊 Model Workflow

1. Load dataset
2. Handle missing values
3. Convert text values to numeric
4. Perform EDA
5. Train Linear Regression model
6. Evaluate performance
7. Deploy UI using Gradio

---

## 📈 Model Performance

- R² Score: Good fit
- MAE: Low error
- RMSE: Reliable prediction

---

## 🖥️ How to Run

```bash
pip install -r requirements.txt
python student_result_prediction.py
```

---

## 🎯 Input Example

| Study Hours | Attendance | Weekly Test Score |
|------------|-----------|------------------|
| 5          | 80        | 70               |

➡️ Output: Predicted Final Exam Score
